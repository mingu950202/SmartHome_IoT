# SmartHome_IoT

아두이노와 MYSQL를 이용한 스마트홈 프로그램입니다.

# 프로젝트 정보
+ 설치
  + Visual Studio
  + Arduino IDE
  + MY SQL
+ 사용 방법
  + 각 센서값에 맞는 라이브러리 다운로드
  + 아두이노 각 센서 연결
  + TempHumid.ino 업로드
  + VisualStudio 이용하여 FINAL.sln 실행
  + MY SQL에 센서 값 저장 필요 시 DB생성 및 Table 생성
  + FINAL.sln MainView에서 사용자 아두이노 포트에 맞게 코드 변경
  + Ctrl + F5
